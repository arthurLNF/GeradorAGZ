salvar()

salvar(string  $quebraDeLinha = "\r\n") : void

Salva um arquivo AGZ.

Este método monta o arquivo AGZ com o header, arrecadação e trailler informados. Caso a geração automática de trailler esteja habilitada, o trailler é gerado com base no header e arrecadação, seguindo o layout Febraban Padrão.

Parameters

string $quebraDeLinha

Usado para quebra de linhas do arquivo. Por padrão, é utilizado "\r\n".

\Exception
